Below is a summary of the technolgies used to create this site. With the exception of Adobe Photoshop and Mac OSX (of which there are certainly acceptable alternatives), everything used was open source and/or free. If you really want to dig into any of the pieces, or even see something you want to shamelessly steal, check out all the source here on GitHub. Or if you just want to see how websites are created with Ruby on Rails these days (and what the custom backend management app looks like), check out the screenshots below.

Web/Dev Stack

Mac OSX Lion 10.7.3

Ubuntu Darwin Kernel Version 11.3.0

Rails 3.0.9

Ruby 1.9.3

Apache/2.2.20 (Ubuntu)

Passenger 3.0.11

A Dell in my living room for hosting (don't tell Comcast Verizon)

Pingdom for when my dynamic IP changes

Gems

Whenever for dynamic Rails cron job management

db2fog for automatic DB backups to Amazon S3 Cloud Storage

thinking_sphinx for article indexing, searching and sphinx daemon management

paperclip for easy model image attachment

oauth for Twitter OAuth authentication

capistrano for code deployment

Javascript

jQuery 1.4.1

Galleria 1.2.6 for image galleries and shadowboxing

CollagePlus for image collages

Tweet.js for twitter feed display

TinyMCE for admin Content Management System

jCrop for UI based image cropping in the admin
